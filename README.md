# personal-portfolio-website

1. Create a Project Folder: Make a folder (e.g., portfolio-project) and save your HTML file as index.html inside it.
2. Build the Structure: Use semantic HTML tags like <header>, <nav>, <main>, <section>, and <footer> to organize the page.
3. Navigation Bar: Add a menu with links to sections like About, Projects, and Contact. These links should scroll to the corresponding sections on the page.
4. Hero & About Section: Include a welcome message, a profile picture (stored in an images folder), and a short bio.
5. Projects Section: List 2-3 sample projects with brief descriptions.
6. Skills Section: Create a table listing technical skills (like HTML, CSS, JavaScript) along with learning levels.
7. Contact Section: Add a form with fields for Name, Email, and Message. Include labels, placeholders, and make the fields required.
